# 🚀 Budda Dileep — Personal Portfolio

A fully responsive personal portfolio website built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies.

🔗 **Live Site:** [buddadileep.vercel.app](https://buddadileep.vercel.app)

---

## 👤 About

Portfolio of **Budda Dileep**, a Mechanical Engineering student at SRKR Engineering College (2023–2027), passionate about CAD design, product development, and additive manufacturing.

---

## ✨ Features

- ⌨️ **Typing animation** on hero section text
- 🖼️ **Photo slide-in** animation after text completes
- 📱 **Fully responsive** — mobile, tablet & desktop
- 🍔 **Full-screen hamburger menu** with photo panel (desktop)
- 🎨 **Scroll reveal animations** on every section
- 💫 **Hover transitions** on all cards and elements
- 📧 **Functional contact form** via mailto
- 📞 **Phone/WhatsApp modal** on number click
- 📄 **Downloadable resume** embedded in the page
- 🔗 **Certificate & project links** to Google Drive
- 🌊 **Floating photo** with animated dashed ring

---
